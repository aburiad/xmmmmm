# Bangla Fonts Directory

## Required Font File

Place the Bangla font file here:
- **Filename:** `NotoSansBengali-Regular.ttf`

## Download Font

1. Visit: https://fonts.google.com/noto/specimen/Noto+Sans+Bengali
2. Click "Download family"
3. Extract the ZIP file
4. Copy `NotoSansBengali-Regular.ttf` to this directory

## File Path

```
/wp-content/plugins/question-paper-pdf-generator/fonts/NotoSansBengali-Regular.ttf
```

## Alternative Fonts

You can also use other Bangla fonts:
- Kalpurush
- SolaimanLipi
- Siyam Rupali

Just rename the file to match the font name in the PDF generator class.
